# iHire.ai - AI Labor Marketplace

The marketplace for AI labor — jobs, workers, tools, and automations.

## Tech Stack
- React + Vite + Tailwind CSS v4
- Supabase (Auth, Database, Edge Functions)
- Stripe (Payments)
- Vercel (Deployment)

## Getting Started
```bash
npm install
cp .env.example .env  # fill in your keys
npm run dev
```
